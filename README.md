# Sistemi i Menaxhimit të Kinemasë (C++)

Ky projekt është realizuar si pjesë e detyrës për menaxhimin e kodit burimor dhe bashkëpunimin në GitHub.

## Ekipi
* **Sara Maliqi** - Lideri i projektit, (Moduli i  Filmave)
* **Rida Hajrullahu** - Zhvillues (Moduli i Ulseve)  

## Përshkrimi
Programi lejon menaxhimin e thjeshtë të një kinemaje, duke përfshirë listimin e filmave dhe rezervimin e ulëseve në sallë përmes një ndërfaqeje në konsolë.

## Si të ekzekutohet
1. Klononi repository: `git clone [https://github.com/saramaliqi/Grupi_67.git]`
2. Kompajloni kodin: `g++ main.cpp -o kinemaja`
3. Ekzekutoni programin: `./kinemaja`

## Workflow i përdorur
Ne kemi përdorur **GitFlow** të thjeshtuar:
- `main` branch për versionin final.
- `feature-` branches për zhvillimin e moduleve.
- **Pull Requests** për qdo bashkim kodi(merge).
